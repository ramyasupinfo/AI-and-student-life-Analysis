# AI and Student life Analysis 
**1. Project Title :**
                   AI and Student life Analysis  - This Project gives insights about different AI tools which is used by students for different purposes and its impact on their lives and scores.

**2. Tech Stack :**     

        The visualisations was built using the following tools
      📈 - Data tranforming, Cleaning and Visualization using pandas,matplolib and seaborn libraries.
 
**3. Data Source :** 

      Source : The dataset  has been taken from the "kaggle" platform which was under "cc Public Domain".
